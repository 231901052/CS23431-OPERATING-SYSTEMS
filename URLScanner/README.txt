# URL Scanner Project

## How to Run

1. Make sure you have Java JDK installed.
2. Download and add JFreeChart jar to your classpath.
3. Compile all java files inside src/ :
   javac -cp "lib/*" src/*.java
4. Run the main program:
   java -cp "lib/*:src/" URLScannerApp

Make sure you download JFreeChart jars and place them inside /lib.
